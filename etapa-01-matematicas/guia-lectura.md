# Etapa 1: Matemáticas Fundamentales

Duración: 2 meses | Prerequisito: ninguno

---

## Álgebra Lineal

### Orden de estudio
1. **3Blue1Brown — Essence of Linear Algebra** (YouTube, gratuito) — ver completo antes de leer
2. **Mathematics for Machine Learning** (Deisenroth) — Capítulos 2, 3, 4
3. **MIT OCW 18.06** (Gilbert Strang) — para profundidad operacional

### Temas
- [ ] Vectores: suma, escalar, norma L1/L2, producto punto
- [ ] Matrices: multiplicación, transpuesta, determinante, inversa
- [ ] Espacios vectoriales: base, dimensión, span, independencia lineal
- [ ] Transformaciones lineales
- [ ] Eigenvalores y eigenvectores
- [ ] SVD (Singular Value Decomposition)
- [ ] PCA como aplicación de SVD

---

## Cálculo

### Orden de estudio
1. **Mathematics for Machine Learning** — Capítulos 5, 6
2. **Khan Academy — Multivariable Calculus** (gratuito, para reforzar)

### Temas
- [ ] Derivadas (repaso)
- [ ] Derivadas parciales
- [ ] Gradiente — dirección de mayor crecimiento
- [ ] Regla de la cadena (crítico para backpropagation)
- [ ] Jacobiano
- [ ] Hessiano
- [ ] Optimización: mínimos y máximos

---

## Probabilidad y Estadística

### Orden de estudio
1. **Introduction to Statistical Learning (ISLR)** — Capítulos 1, 2
2. **Mathematics for Machine Learning** — Capítulo 6

### Temas
- [ ] Variables aleatorias discretas y continuas
- [ ] Distribuciones: Normal, Bernoulli, Binomial, Poisson
- [ ] Esperanza, varianza, covarianza
- [ ] Teorema de Bayes
- [ ] MLE (Maximum Likelihood Estimation)
- [ ] MAP (Maximum A Posteriori)
- [ ] Correlación de Pearson

## Señales de que dominas la etapa

- Puedes derivar la solución closed-form de regresión lineal: `β = (XᵀX)⁻¹Xᵀy`
- Puedes calcular eigenvalores a mano para una matriz 2×2
- Puedes demostrar el Teorema de Bayes desde probabilidades conjuntas
- Entiendes intuitivamente qué hace el gradiente en un espacio N-dimensional
